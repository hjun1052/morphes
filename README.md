# 🧠 MORPHES — AI Prompt Engineering Assistant  
**URL:** [https://morphes.codz.me](https://morphes.codz.me)

MORPHES는 사용자의 요구를 대화로 이해하고, 수많은 설문과 피드백 과정을 거쳐 **최적의 AI 프롬프트를 자동 설계**하는 웹 서비스입니다.  
ChatGPT, Claude, Gemini, Perplexity 등 다양한 AI 서비스에 바로 사용할 수 있는 완성된 프롬프트를 생성합니다.

---

## 🚀 주요 기능

### 1. 대화형 프롬프트 설계
- 사용자가 “무엇을 하고 싶은지” 설명하면 MORPHES가 대화를 시작합니다.  
- 목적, 스타일, 출력 형식, 톤 등을 설문 형식으로 세밀하게 수집합니다.  
- 여러 번의 대화를 통해 최적의 프롬프트를 함께 다듬습니다.

### 2. 설문 기반 세부 요구 파악
- 단계별 맞춤 설문을 제시합니다.
- 선택형(multiple choice) / 입력형(input) 설문을 자유롭게 혼합합니다.
- 설문은 프롬프트 목적, AI 역할, 출력 구조, 표현 스타일 등을 세분화해 질문합니다.

### 3. 2개의 후보 프롬프트 제안
- 충분한 정보가 모이면 완성도 높은 2개의 영어 프롬프트 후보를 제안합니다.
- 각 프롬프트는 바로 AI 서비스에 붙여넣을 수 있는 수준으로 작성됩니다.

### 4. 피드백 기반 반복 개선
- 사용자가 하나를 선택하면, 대화를 통해 이를 반복적으로 수정합니다.
- 문체, 논리, 구체성, 표현력 등을 개선하여 점점 완성도 높은 프롬프트를 만들어갑니다.

### 5. 최종 확정
- 사용자가 만족하면 이를 채팅 서비스에 바로 보낼 수 있도록 제공합니다.
